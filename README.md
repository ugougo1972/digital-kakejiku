# digital-kakejiku

静かに空間を観測し、
時間と環境の変化を記録する、
据置型デジタル観測装置。

本プロジェクトは、
ESP32 / Google Apps Script / Gemini / E-Paper 等を用いて、

- 環境状態
- 空気感
- 季節感
- 滞在感
- AI生成文
- 時系列変化

を長期的に蓄積・可視化することを目的としています。

---

# コンセプト

digital-kakejiku は、
単なる情報表示端末ではありません。

本装置は：

- 静けさ
- 余白
- 時間性
- 空気感
- 緩慢な変化

を重視した、
「個人的観測装置」です。

高FPS・派手な演出・常時アニメーションよりも、

- 静止
- 季節変化
- 文脈
- 微細な環境変動

を重視します。

---

# 現時点での構成案

現時点では、
以下構成を中心に検討を進めています。

```text
Sensors
↓
ESP32-S3
↓ HTTPS
Google Apps Script
↓
Google Spreadsheet
↓
Gemini API
↓
E-Paper / LCD
```

## AI生成コンテキスト

# 表示思想

現在は以下2案を比較中。

## LCD案

- 高解像度UI
- 慣性スクロール
- LVGLベース
- 動的レイアウト

## E-Paper案（現時点有力）

- 非発光
- 低頻度更新
- 電子的掛軸
- 静的表現
- 長時間表示

思想的には E-Paper を優先しています。
