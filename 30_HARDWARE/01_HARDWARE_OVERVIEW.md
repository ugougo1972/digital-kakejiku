# digital-kakejiku ハードウェア概要

最終更新: 2026-05-31

---

# 1. 目的

本ドキュメントは、digital-kakejiku（据置型日めくり観測器）のハードウェア構成を整理する。

本装置は、環境情報を長期観測し、Google Apps Script / Spreadsheet / Gemini API と連携しながら、7.5インチE-Paperへ静的・低頻度に表示する据置型観測装置である。

---

# 2. 全体構成

```text
センサー群
↓
XIAO ESP32S3 Plus
↓ Wi-Fi / HTTPS POST
Google Apps Script
↓
Google Spreadsheet
↓
Gemini API
↓
7.5inch 800×480 E-Paper
```

# 3. MCU

## 採択

- XIAO ESP32S3 Plus

## GPIO方針

- XIAO本体の2.54mmピッチヘッダーに出ている端子のみ使用する
- 1.27mmピッチ相当の細かいランドは使用しない
- GPIO不足対策としてMCP23017を採用する
- ロータリーエンコーダ、ボタン類、補助GPIOは原則としてMCP23017側へ逃がす

---

# 4. 電源構成

## 採択済み部材

- 18650 Li-ion
- 18650電池ホルダー
- IP5306系 18650充電・5V電源管理モジュール
- TPS63802 昇降圧DC/DC
- P-MOSFET逆流防止回路

## 電源ブロック図

```text
USB-C入力
↓
IP5306系 18650充電・5V電源管理
↓ 5V
TPS63802
↓ 3.3V
P-MOSFET逆流防止回路
↓
3.3V電源バス
↓
XIAO ESP32S3 Plus
センサー群
MCP23017
RTC
```

## 電源方針

- 据置型のため、通常稼働はUSB給電を基本とする
- USB給電喪失時のみ18650からのバックアップ給電へ移行する
- UPS的動作を目指す
- 手動切替は行わず、ヒューマンエラーを機械的に排除する
- 既存の3.3V前提の開発環境を維持する
- XIAOへは3.3V系から給電する
- XIAOのUSB接続時に外部3.3V系からPC側へ逆流しないよう、P-MOSFET逆流防止回路を設ける

## 注意事項

- IP5306モジュールの端子名、出力仕様、常時出力可否は実モジュールで確認する
- TPS63802の入出力コンデンサ、EN端子または制御端子の扱いは使用モジュール仕様に従う
- P-MOSFETの型番、向き、ゲート処理、プル抵抗値は未確定
- 3.3Vバスの実負荷電流はセンサー統合後に確認する

# 5. 表示装置

## 採択

- 7.5inch 800×480 E-Paper
- XIAO ePaper Breakout Board V2

## 方針

- 高FPS表示は不要
- 静的表示、低頻度更新、余白、時間性を重視する
- 日めくりカレンダー的な表示を基本方向とする

---

# 6. センサー構成

## 採択済み

| センサー | 用途 | 接続方針 | 備考 |
|---|---|---|---|
| SCD41 | CO₂ / 温湿度 | I2C | SCD40ではなくSCD41を採択 |
| SGP41 | VOC / NOx | I2C | Sensirion系でSCD41と役割分担 |
| SPS30 | 粒子状物質 | I2C想定 | 内蔵ファン前提、外付けファン不要 |
| LTR390 | 照度 / UV | I2C | 既存部材流用 |
| BME680 | 温湿度 / 気圧 / 補助ガス | I2C | BME280系から更新 |
| HLK-LD2410C | 人感 | OUT 1本を基本 | UART詳細取得は後日拡張または保守用 |

---

# 7. GPIO拡張

## 採択

- MCP23017

## 用途

- ロータリーエンコーダ
- ボタン類
- 補助GPIO

## 方針

XIAO本体のGPIOは、E-Paper、microSD、I2C、必要最小限の制御線へ優先的に割り当てる。
操作系はMCP23017へ逃がす。

---

# 8. ストレージ

## 採択

- microSD

## 接続方針

- SPI接続
- E-PaperとSCK / MOSI / MISOを共有
- microSD専用CSを1本使用

---

# 9. RTC

## 採択方針

- I2C接続のRTCモジュールを搭載する

## 未確定

- RTC型番
- 電池形式
- AT24有無
- INT / SQW割り込み使用有無

---

# 10. 筐体・通気方針

- 底面解放＋高さ調整脚で通気を確保する
- 天板または上部側面に排気口を設ける
- SCD41 / SGP41 / BME680 は底面開放部近く、熱源や内部VOC源から離して配置する
- SPS30は吸入口・排出口を塞がず、外気に近い位置へ配置する
