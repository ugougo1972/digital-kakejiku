# DECISION_0006: XIAO D11〜D19側面ランド取り出し方針

作成日: 2026-06-25  
対象プロジェクト: digital-kakejiku  
分類: HARDWARE  
状態: ADOPTED

---

# 1. 決定事項

XIAO ESP32S3 Plus の追加GPIO取り出しについて、以下を採択する。

```text
XIAO ESP32S3 Plus は1台構成を維持する。
ePaperドライバーボード装着時に使用困難となる裏面ランドは使用禁止とする。
D11〜D19に相当する1.27mm側面ランドは使用解禁とする。
D11〜D19は、信号線＋GNDのペアとして本体基板へ接続する。
```

---

# 2. 採択方針

## 2.1 MCU構成

| 項目 | 決定 |
|---|---|
| MCU | Seeed XIAO ESP32S3 Plus |
| MCU台数 | 1台 |
| 2台構成 | 初号機では不採択。フォールバックとして記録のみ |
| ePaper接続 | XIAO ePaper Breakout V2 / ePaperドライバーボード経由 |
| 裏面ランド | 使用禁止 |
| 1.27mm側面ランド | 使用解禁 |
| D11〜D19 | 1.27mm側面ランドに相当し、使用可 |

## 2.2 ePaperドライバーボード配置

7.5inch ePaper本体には長めのリボンケーブルが付属しているため、ePaperドライバーボードおよびXIAOをePaper本体直近に固定する必要はない。

したがって、以下を採択する。

```text
ePaper本体
  ↓ 付属リボンケーブル
ePaperドライバーボード + XIAO ESP32S3 Plus
  ↓ 短距離接続
大型本体基板
```

この配置により、D11〜D19側面ランドから大型本体基板までの配線長を短くできる。

---

# 3. D11〜D19取り出し用ミニ基板

## 3.1 役割

D11〜D19取り出し用ミニ基板は、部品実装基板ではなく、以下に特化したアダプタとして扱う。

- D11〜D19側面ランドの信号取り出し
- XIAO GNDの取り出し
- 小規模GNDバスの形成
- 信号線＋GNDのJST 2ピン化
- 本体基板への短距離接続

## 3.2 載せるもの

| 対象 | 方針 |
|---|---|
| D11〜D19信号配線 | 載せる |
| XIAO由来GND | 載せる |
| 小規模GNDバス | 載せる |
| JST 2ピンコネクタ | 載せる |
| GPIO番号ラベル | 推奨 |

## 3.3 原則載せないもの

| 対象 | 方針 | 理由 |
|---|---|---|
| プルアップ抵抗 | 原則載せない | D11〜D19用途未確定のため |
| プルダウン抵抗 | 原則載せない | D11〜D19用途未確定のため |
| RCフィルタ | 原則載せない | ADC用途確定後に判断するため |
| 大型コンデンサ | 載せない | ミニ基板へ機械的・電気的負荷をかけないため |
| 電源系部品 | 載せない | 大電流経路から分離するため |
| 大電流GND配線 | 載せない | 信号用GNDと電源帰路を混在させないため |

# 4. JST 2ピン接続方針

D11〜D19は、各GPIOごとにJST 2ピンで取り出す。

| JST | Pin1 | Pin2 | ケーブル |
|---|---|---|---|
| JST-D11 | D11 | GND | D11 + GND 撚り線 |
| JST-D12 | D12 | GND | D12 + GND 撚り線 |
| JST-D13 | D13 | GND | D13 + GND 撚り線 |
| JST-D14 | D14 | GND | D14 + GND 撚り線 |
| JST-D15 | D15 | GND | D15 + GND 撚り線 |
| JST-D16 | D16 | GND | D16 + GND 撚り線 |
| JST-D17 | D17 | GND | D17 + GND 撚り線 |
| JST-D18 | D18 | GND | D18 + GND 撚り線 |
| JST-D19 | D19 | GND | D19 + GND 撚り線 |

各JSTにはGPIO番号を明示する。

```text
JST-D11: Pin1=D11 / Pin2=GND
JST-D12: Pin1=D12 / Pin2=GND
...
JST-D19: Pin1=D19 / Pin2=GND
```

---

# 5. GND接続方針

## 5.1 ミニ基板側

D11〜D19取り出し用ミニ基板には、XIAO由来のGNDを取り込み、小規模GNDバスを設ける。

```text
XIAO GND
  ↓
D11〜D19取り出し用ミニ基板 GNDバス
  ├─ JST-D11 GND
  ├─ JST-D12 GND
  ├─ ...
  └─ JST-D19 GND
```

このGNDは、信号の戻り道、基準電位、ノイズ対策のために使用する。

## 5.2 本体基板側

本体基板側JSTのGNDは、本体基板のGNDバスへ直結する。

```text
JST-D11 Pin2 GND
  ↓
本体基板 GNDバス
```

ただし、以下は行わない。

- SPS30の大電流帰路をD11〜D19用GNDへ流す
- DC-DCの大電流帰路をD11〜D19用GNDへ流す
- ePaper更新時の電源帰路をD11〜D19用GNDへ流す
- D11〜D19用GNDを電源GNDの数珠つなぎ経路にする

# 6. 抵抗・ノイズ対策の扱い

D11〜D19の用途が未確定であるため、初期段階ではミニ基板上に抵抗を実装しない。

採択方針。

```text
D11〜D19ミニ基板は、信号線とGNDの取り出しに特化する。
抵抗、RC、プルアップ、プルダウンは、用途確定後に本体基板側または信号源側で設計する。
```

将来の考え方は以下とする。

| 用途 | 抵抗・部品の基本配置 |
|---|---|
| GPIO出力 | 信号源側に直列抵抗を検討 |
| GPIO入力 | 受け側入力近くにプルアップ/プルダウンを検討 |
| SPI CS | 非選択固定用プルアップを検討 |
| 割り込み線 | 受け側でプルアップを検討 |
| ADC | 分圧抵抗とRCフィルタを入力近傍に検討 |
| I2C | バス全体で1組のプルアップ。重複に注意 |

# 7. 信号品質方針

D11〜D19は、長距離空中配線を前提にしない。  
ただし、ノイズ対策として以下を維持する。

| 対策 | 方針 |
|---|---|
| 信号線とGNDのペア化 | 採択 |
| 信号線＋GNDの撚り線 | 採択 |
| JST 2ピン単位のラベル明示 | 採択 |
| DC-DC電源線との並走回避 | 推奨 |
| SPS30ファン電源線との並走回避 | 推奨 |
| ePaper電源線との並走回避 | 推奨 |
| 直列抵抗 | 必要時に用途確定後対応 |

# 8. 電源関連の同時決定

## 8.1 SPS30電源

SPS30の動作電圧は以下で確定する。

```text
SPS30動作電圧: 3.5V〜5.0V
供給元: 5V BUS
```

注意点。

- SPS30は3.3V BUSへ接続しない。
- SPS30は5V BUS負荷として扱う。
- I2Cプルアップ先はXIAOに合わせて3.3V系に統一する。

## 8.2 逆流防止Pch MOSFET

初号機では、手元在庫を優先し、DMG3415Uを採択する。

| 項目 | 決定 |
|---|---|
| 初号機採用品 | DMG3415U |
| 用途 | 5V BUS逆流防止用 Pch MOSFET |
| 代替候補 | DMG2305UX-13 |
| ヒートシンク | 原則不要 |
| 放熱方針 | 銅箔放熱と実測確認を優先 |
| 確認対象 | SPS30動作時、ePaper更新時、Wi-Fi送信時、BATTERY_MODE時 |

# 11. 決定理由

- ePaperドライバーボード装着時、XIAO裏面ランドは物理的に使用困難である。
- 1.27mm側面ランドはD11〜D19に相当し、追加GPIO取り出し先として利用できる。
- ePaper本体付属の長めリボンケーブルにより、ドライバーボード＋XIAOを本体基板近傍へ配置できる。
- D11〜D19の長距離空中配線を避けられるため、信号品質と機械的安定性が改善する。
- 各信号をGNDと2線ペア化することで、戻り電流経路を明確化できる。
- JST 2ピン単位にすることで、GPIO番号の明示、誤配線防止、保守性が向上する。
- 用途未確定段階で抵抗類をミニ基板へ載せると、後工程で不整合が生じる可能性がある。
